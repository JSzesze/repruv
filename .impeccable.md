## Design Context

### Users

People who need clean, portable Markdown from a public webpage or X post without creating an account. The core audience includes writers, researchers, developers, and people feeding web material into notes or AI workflows. They arrive with a URL and want a trustworthy result immediately.

### Brand Personality

Calm, precise, immediate. Repruv should feel like a well-made public utility: quietly confident, fast, and candid about caching and public URLs.

### Aesthetic Direction

An almost unstyled black utility: no hero heading, full-page texture, custom font, or explanatory copy. A tiny upper-left REPRUV wordmark with one orange status dot provides site identity. The initial view contains one quiet, compact URL input and its submit control, backed by a localized Aceternity-style matrix of orange dots that pulse independently in opacity and bloom when bright. The dot field is the visual feature; the input itself uses warm-black surfaces, a low-contrast neutral border even while focused, and a muted arrow that becomes orange only through interaction. Orange is the sole secondary color and stays restrained to the brand dot, input halo, focus, submit control, and quiet interaction states. Results reveal only the document title, cache metadata, copy/download actions, and Markdown. Avoid marketing copy, ambient decoration, dashboards, feature lists, API promotion, cards, or ornamental imagery. The primary experience should work equally well on desktop and mobile with strong keyboard and focus behavior. Respect reduced-motion with a static dot field.

### Design Principles

1. The URL is the only required input; make the path from paste to Markdown feel instantaneous.
2. Make system state legible but quiet: processing, cached, copied, and failed should never compete with the content.
3. Earn every element. If it does not help users convert, inspect, copy, or download, remove it.
4. Use proportion, hairlines, and typography for polish; decoration stays confined to the input module.
5. Keep the public-service contract visible: free, anonymous, cached, and intended only for public URLs.
